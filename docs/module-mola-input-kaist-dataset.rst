========================================
Module: mola-input-kaist-dataset
========================================

Reads from a `KAIST SLAM dataset`_, including:
 - Two 3D lidars (Velodyne VLP16)
 - Two 2D lidars (SICK)
 - Stereo camera pair.
 - Wheel odometry.
 - FOG.
 - Altimeter.


.. _`KAIST SLAM dataset`: http://irap.kaist.ac.kr/dataset/
